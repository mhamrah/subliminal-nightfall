# Subliminal Nightfall

A beautiful dark theme inspired by Rose Pine with carefully calibrated colors for optimal readability and visual comfort.

## Features

- **Deep purple-black background** (#191724) for reduced eye strain
- **Nord red accents** (#bf616a) for errors and important highlights
- **Properly calibrated bright/dim variants** for better terminal contrast
- **Semantic color usage** - colors have meaning (red = errors, green = success, yellow = warnings)
- **Optimized for programming** - Swift, Rust, TypeScript, and more

## Color Palette

Base colors with bright and dim variants:

| Color   | Base    | Bright  | Dim     | Usage                    |
|---------|---------|---------|---------|--------------------------|
| Red     | #bf616a | #e2848d | #85434a | Errors, deletions        |
| Green   | #a9cfa4 | #ccf2c7 | #769072 | Success, additions       |
| Yellow  | #ffe2a9 | #ffffcc | #b29e76 | Warnings, modifications  |
| Blue    | #6699cc | #89bcef | #476b8e | Info, titles, predictive |
| Magenta | #f1a5ab | #ffc8ce | #a87377 | Attributes, emphasis     |
| Cyan    | #5fb3b3 | #82d6d6 | #427d7d | Focus borders, operators |

**Background**: #191724 (deep purple-black)  
**Foreground**: #e0def4 (soft white)

## Installation

1. Open Cursor or VS Code
2. Go to Extensions (Cmd+Shift+X / Ctrl+Shift+X)
3. Search for "Subliminal Nightfall"
4. Click Install
5. Press Cmd+K Cmd+T (or Ctrl+K Ctrl+T) and select "Subliminal Nightfall"
